
The following commands are used to work with this project:

*   **`bundle install`**: Installs the required Ruby gems from the `Gemfile`.
*   **`bundle exec jekyll serve`**: Builds the site and serves it locally at `http://localhost:4000`. The site will be automatically regenerated when changes are made.
*   **`bundle exec jekyll build`**: Builds the site and outputs the static files to the `_site` directory.
