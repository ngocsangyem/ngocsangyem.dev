
The project is built with the following technologies:

*   **Jekyll:** A static site generator written in Ruby.
*   **Hydejack:** A Jekyll theme.
*   **Ruby:** The programming language used by Jekyll.
*   **Bundler:** A dependency manager for Ruby.
*   **Sass:** A CSS preprocessor.
*   **JavaScript:** Used for client-side interactivity. The theme uses a modular approach, loading features like dark mode, drawer, push-state, and table of contents based on browser capabilities. It also uses Webpack for bundling.
*   **Markdown:** Used for writing blog posts and pages.
*   **KaTeX:** For mathematical typesetting.
