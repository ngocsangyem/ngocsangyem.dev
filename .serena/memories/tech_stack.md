# Technology Stack

## Core Framework
- **Jekyll 4.3** - Static site generator
- **Hydejack Theme** - Premium Jekyll theme (local version in `#jekyll-theme-hydejack/`)
- **Ruby** - Programming language for Jekyll
- **Bundler** - Ruby dependency management

## Key Dependencies (from Gemfile)
- `jekyll-theme-hydejack` - Main theme (local path)
- `kramdown-math-katex` - Math formula rendering
- `duktape` - JavaScript runtime for Ruby
- `webrick` - Web server

## Jekyll Plugins
- `jekyll-default-layout`
- `jekyll-feed` - RSS/Atom feeds
- `jekyll-optional-front-matter`
- `jekyll-paginate` - Blog pagination
- `jekyll-readme-index`
- `jekyll-redirect-from`
- `jekyll-relative-links`
- `jekyll-seo-tag` - SEO optimization
- `jekyll-sitemap` - XML sitemap generation
- `jekyll-titles-from-headings`
- `jekyll-include-cache` - Performance optimization
- `jekyll-last-modified-at` - Last modified dates
- `jekyll-compose` - Content creation helpers

## Frontend Technologies
- **HTML5** - Markup
- **CSS3/Sass** - Styling (custom styles in `_sass/`)
- **JavaScript** - Client-side functionality
- **KaTeX** - Math rendering
- **Responsive Design** - Mobile-first approach

## Build Tools
- **Sass** - CSS preprocessing (compressed output)
- **Jekyll Build System** - Static site generation
- **Kramdown** - Markdown processing with math support
