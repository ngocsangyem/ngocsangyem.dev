## Quick Config Changes
**To enable Service Worker:**
```yaml
offline:
  enabled: true
  cache_version: 13
```

**To add analytics:**
Edit `_includes/my-body.html` or configure in `_includes/body/analytics.html`

**To change CSS inlining:**
```yaml
hydejack:
  no_inline_css: false  # Enable inlining (default)
  no_page_style: false  # Enable per-page colors
```

**To disable search:**
```yaml
hydejack:
  no_search: false  # false = enabled in production
```

## Build Commands
```bash
# Development
bundle exec jekyll serve

# Production (enables search, compression)
JEKYLL_ENV=production bundle exec jekyll build

# With related posts (LSI)
JEKYLL_ENV=production bundle exec jekyll build --lsi

# Profile build timing
bundle exec jekyll build --profile
```

## Critical Paths to Performance
1. Reduce HTML size - Already compressed
2. Inline critical CSS - Already inlined
3. Defer non-critical CSS - Already deferred
4. Optimize images - WebP + responsive variants ready
5. Lazy load images - Consider adding `loading="lazy"`
6. Enable caching - Include-cache enabled; SW available
7. Minify assets - Already minified
